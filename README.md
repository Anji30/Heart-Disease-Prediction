# Heart Disease Prediction Using Machine Learning

## Project Overview

This project predicts whether a patient has heart disease based on
medical parameters using Machine Learning algorithms.

## Objective

The objective of this project is to analyze patient health data and
predict the likelihood of heart disease using classification algorithms.

## Technologies Used

-   Python
-   Pandas
-   NumPy
-   Scikit-learn
-   Matplotlib

## Dataset

The project uses a dataset named **heart.csv** which contains medical
attributes such as:

-   Age
-   Sex
-   Chest pain type
-   Blood pressure
-   Cholesterol level
-   Maximum heart rate
-   Exercise induced angina
-   Target (Heart disease presence)

## Machine Learning Algorithms Used

-   Logistic Regression
-   Decision Tree
-   Random Forest
-   Support Vector Machine (SVM)
-   K-Nearest Neighbors (KNN)

## Project Structure

Heart-Disease-Prediction │ ├── heart.csv ├── heart_prediction.py ├──
graphs.py └── README.md

## How to Run the Project

1.  Install required libraries

pip install pandas scikit-learn matplotlib

2.  Run the main program

python heart_prediction.py

3.  The model will train using the dataset and predict whether heart
    disease is present.

## Output

The system displays: - Model accuracy - Prediction result (Heart Disease
or No Heart Disease)
